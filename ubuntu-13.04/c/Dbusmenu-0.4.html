Class: DbusmenuMenuitem

Known subclasses: DbusmenuMenuitemProxy

Subclass of: GObject

This is the #GObject based object that represents a menu item. It gets created the same on both the client and the server side and libdbusmenu-glib does the work of making this object model appear on both sides of DBus. Simple really, though through updates and people coming on and off the bus it can lead to lots of fun complex scenarios.

Properties

gint"id"construct-only

Signals

void"realized"(DbusmenuMenuitem *menuitem)

Emitted when the initial request for properties is complete on the item. If there is a type handler configured for the "type" parameter that will be executed before this is signaled.

void"show-to-user"(DbusmenuMenuitem *menuitem, guint arg1)

Signaled when the application would like the visualization of this menu item shown to the user. This usually requires going over the bus to get it done.

void"child-added"(DbusmenuMenuitem *menuitem, GObject arg1, guint arg2)

Signaled when the child menuitem has been added to the parent.

gboolean"event"(DbusmenuMenuitem *menuitem, gchar* arg1, GVariant arg2, guint arg3)

Emitted when an event is passed through. The event is signalled after handle_event is called.

void"property-changed"(DbusmenuMenuitem *menuitem, gchar* arg1, GVariant arg2)

Emitted everytime a property on a menuitem is either updated or added.

gboolean"about-to-show"(DbusmenuMenuitem *menuitem)

Emitted when the submenu for this item is about to be shown

void"child-removed"(DbusmenuMenuitem *menuitem, GObject arg1)

Signaled when the child menuitem has been requested to be removed from the parent. This signal is called when it has been removed from the list but not yet had #g_object_unref called on it.

void"item-activated"(DbusmenuMenuitem *menuitem, guint arg1)

Emitted on the objects on the server side when they are signaled on the client side.

void"child-moved"(DbusmenuMenuitem *menuitem, GObject arg1, guint arg2, guint arg3)

Signaled when the child menuitem has had its location in the list change.
